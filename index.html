Powerful Commands

/play [song] Plays any song or playlist from your favorite platforms.
/queue View and manage the upcoming music tracks.
/skip Instantly jump to the next song in the queue.
/nowplaying Get detailed info about the currently playing track.
/bassboost Enhance the low frequencies for a powerful sound.
/nightcore Speed up and pitch up your tracks instantly.
/8d Experience an immersive 8D spatial audio effect.
/prefix Change the bot prefix for your server.
/setup Create a dedicated music request channel.

How to Setup

1

Configure Environment

Open the .env file and add your DISCORD_TOKEN and CLIENT_ID.

2

Install Dependencies

Open your terminal and run npm install to set up the engine.

3

Start Streaming

Execute npm start and your Synapse Beats instance is live!
